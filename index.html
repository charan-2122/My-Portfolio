💬 What is Kirchhoff's Voltage Law and how do I apply it in a circuit with 3 loops?
Kirchhoff's Voltage Law (KVL) states that the algebraic sum of all voltages around any closed loop in a circuit must equal zero.

For 3-loop circuits: define loop directions → apply KVL to each loop → solve the resulting system of 3 equations simultaneously.

Key formula: ΣV = 0 for each loop
